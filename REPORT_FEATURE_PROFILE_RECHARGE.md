## 一、需求概述

在已有功能基础上，新增个人中心页面和账户充值功能。

## 二、新增内容

### 2.1 后端路由

| 路由 | 方法 | 功能 |
|------|------|------|
| `/profile` | GET | 查看用户资料（ID/用户名/邮箱/手机/余额） |
| `/recharge` | POST | 余额充值 |

### 2.2 数据库变更

```sql
ALTER TABLE users ADD COLUMN balance REAL DEFAULT 0;
```

### 2.3 页面模板

| 文件 | 说明 |
|------|------|
| `profile.html` | 个人中心 + 充值表单 |

## 三、初始安全缺陷

| 缺陷 | 说明 |
|------|------|
| 水平越权 IDOR | `/profile?user_id=X` 可查看任意用户 |
| 无登录校验 | 未登录也可访问 profile 和 recharge |
| 负值充值 | `amount=-99999` 可减少余额 |
| SQL 注入 | `WHERE id = {user_id}` f-string 拼接 |
| 充值无 CSRF | 无 Token 验证 |

## 四、修复内容

| 缺陷 | 修复方式 |
|------|---------|
| 水平越权 | `row[1] != username` 身份校验 |
| 登录校验 | 添加 `session.get("username")` 检查 |
| 负值充值 | `amount <= 0` 拒绝 |
| SQL 注入 | 改为参数化查询 |
| CSRF | 充值表单添加 csrf_token |
